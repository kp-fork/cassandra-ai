# 사주 엔진 분석 및 개선점

## 현재 상태 (v1.1.0)

```
lib/saju-engine.ts — Python→TS 포팅, vibe-investing/AIInvestor 참고
```

### 구현된 기능 (v1.2.0)
- ✅ 60갑자 일주 계산 (1900-01-01 기준 갑술 앵커)
- ✅ **4주(년·월·일·시) 완전 계산** ✨NEW
- ✅ 천간·지지 → 오행 변환
- ✅ 오행 상생(生)·상극(克) 관계
- ✅ 오늘 일진 vs 사용자 일간 → 5종 관계
- ✅ 일주별 성격·특징·단점 DB (60종)
- ✅ **주간 운세** (7일 시뮬레이션 + 트렌드 + 베스트/워스트) ✨NEW
- ✅ **월간 운세** (30일 시뮬레이션 + 상승/하락 곡선) ✨NEW
- ✅ **연간 총운** (2026 병오년 세운 나래이티브) ✨NEW
- ✅ 5운 점수 (재물·사업·학업·연애·건강)
- ✅ 선호 오행 계산 (종목 추천용)
- ✅ Redis 캐싱 + 브라우저 localStorage

## 확장 아이디어

### 주간/월간 운세 API
```
GET /api/saju/weekly?birthDate=1990-05-15&birthHour=8
→ 7일치 일진 + 5운 평균 + 최고/최저 날짜
→ 추천 매매 타이밍 (운세 좋은 날)
```

```
GET /api/saju/monthly?birthDate=1990-05-15&birthHour=8
→ 30일치 일진 패턴 + 월간 운세
→ 이번 달 투자 방향 (공격적/방어적)
```

### 시즌 리포트
```
입춘(2/4)·하지(6/21)·추분(9/23)·동지(12/22) 기준
계절별 운세 변화 + 섹터 추천
```

## 참고 사항

현재 엔진은 **일주(日柱) 기준 오행 우세**와 **오늘 일진과의 상생상극**만을 계산합니다.
이는 원본 Python 코드의 `saju_engine.py` 설계 의도와 동일하며,
**"취미와 재미로 살펴보는 간이 사주"**라는 서비스 성격에 부합합니다.

실제 명리학의 정밀한 통변(사주 풀이)을 위해서는 위 개선점을 단계적으로 적용해야 합니다.
