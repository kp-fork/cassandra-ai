# CASSANDRA AI — 소셜 미디어 포스팅

## 링크드인 포스트 (전문가용)

🚀 **코스닥 DART 공시를 실시간으로 분석하는 무료 AI 플랫폼을 만들었습니다.**

CASSANDRA AI는 1,822개 코스닥 종목의 DART 공시를 자동 수집·분석하고, 사이토스케이프 관계망으로 핵심 인물·회사 간 연결고리를 시각화합니다. 인프라 비용은 $0/월입니다.

**핵심 기능:**
- 섹터별 공포·탐욕 지수 (Yahoo Finance × 10 ETF 실시간)
- MU(마이크론) → SK하이닉스 시가 예측 (크로스마켓 회귀, 적중률 71%)
- DART 챗봇: 자연어로 공시 분석 질의
- Cytoscape.js 관계망 + 인물 검색 + 주식셀럽 WIKI

**기술 스택:**
Next.js 15 + TypeScript + Prisma + Neon PostgreSQL + Upstash Redis + Vercel + GitHub Actions

모든 코드는 오픈소스입니다. 스타 하나가 큰 힘이 됩니다 ⭐

🔗 https://github.com/gameworkerkim/cassandra-ai
🌐 https://dart-monitor-pi.vercel.app/quant
